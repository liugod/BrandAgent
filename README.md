# BrandAgent

BrandAgent is a Vercel-ready SaaS MVP scaffold for a brand content operations system. It turns the PRD in `docs/` into an executable product skeleton with:

1. Product spec and acceptance criteria
2. Test case documentation
3. A Next.js application shell for the core SaaS pages
4. Domain helpers and baseline automated tests

## Stack

1. Next.js App Router
2. TypeScript
3. Vitest
4. React Testing Library

## Routes

1. `/` marketing home
2. `/workspace` dashboard
3. `/onboarding` brand initialization
4. `/ingest` source dropbox
5. `/assets` asset center
6. `/calendar` content calendar
7. `/packages/demo` content package detail
8. `/settings` workspace settings

## Commands

```bash
npm install
npm test
npm run build
npm run dev
```
